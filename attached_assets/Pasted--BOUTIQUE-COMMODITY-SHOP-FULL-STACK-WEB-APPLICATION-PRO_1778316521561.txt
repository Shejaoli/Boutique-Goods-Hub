 BOUTIQUE COMMODITY SHOP — FULL-STACK WEB APPLICATION PROMPT

📌 OVERVIEW
Build a complete, professional full-stack commodity shop web application for a boutique store selling everyday goods such as sugar, rice, cooking oil, salt, flour, tomato paste, beans, groundnut oil, and similar items.
The system has two sides:

A rich, mobile-first customer-facing storefront (inspired by the reference designs: clean mobile grocery app UX from images 2, 3, 4, and 5)
A powerful admin dashboard for the owner and staff (inspired by the dashboard designs from images 6 and 7)
An admin login page inspired by the bold curved-background login screen in image 8


I will provide product images. All product image slots must accept uploaded images and display them beautifully in product cards, detail pages, and the admin inventory table.
